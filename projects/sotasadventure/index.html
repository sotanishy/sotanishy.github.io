Sota's Adventure

Sota's Adventure is a simple side-scrolling RPG game, just like Super Mario Bros.

This is one of the biggest and most time-consuming project I have worked on, taking me some 5 months from writing the first line of code to publishing it on GitHub. Although there were some frustrating moments along the way, this is one of my favorite projects.

This game is developed in Java, without any external library or framework (at that time I thought using someone else's code was cheating). Taking advantage of 3rd party code would have allowed me to finish the project much quicker, but I gained a lot from this experience and it surely made me a better coder.

Oh, by the way, the icon for this website (and pretty much everything I use) came from this game :)

Download the game or view the source code on GitHub.

Sota's AdventureはSuper Mario Bros.のような、シンプルな横スクロールRPGゲームです。

コードの最初の1行を書いてからGitHubで公開するまでの期間は約5か月で、私が取り組んだプロジェクトの中で最大かつ最長のものです。途中何度も挫折しそうになりましたが、私のお気に入りのプロジェクトです。

このゲームはJavaで開発しました。外部ライブラリやフレームワークは一切使用しておりません（当時は、他人が書いたコードを使うのはチートだと思っていました）。外部のコードを利用すればこのプロジェクトはずっと早く終わったと思いますが、この経験から多くのことを学ぶことができ、より良いコーダーになることができたと思います。

ところで、このウェブサイト（と私が使うもののほとんど）のアイコンは、このゲームから来ています :)

GitHubでゲームのダウンロードとソースコードの閲覧ができます。
